# hypester

Aggregator of popular social media content from Reddit, Instagram, Telegram, Youtube, VK.
<br/><br/>
Stack: Bloc, Dio, GetIt, DRAW, VK SDK, Youtube APi, Telegram Bot API, Hive, Firebase
